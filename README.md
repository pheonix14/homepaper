# Homepaper

Homepaper is an open-source, highly performant live wallpaper application for Windows, featuring a stunning futuristic UI. It allows you to seamlessly set videos (`.mp4`, `.webm`) or highly interactive web pages (`.html`) directly behind your desktop icons.

**Developed by pheonix14**

## Features
- **Futuristic Frameless UI**: Sleek dark mode with neon glowing effects.
- **Video & HTML Support**: Use looping videos or dynamic web apps as your wallpaper.
- **Easy Management**: Just drop your files into the `papers/` folder and they instantly appear in the app!
- **System Tray Integration**: Runs quietly in the background without cluttering your taskbar.

## Installation
1. Clone this repository.
2. Run `install.bat` to automatically set up the Python environment and dependencies.

## Usage
- Run `run.bat` to start the app silently.
- Open the dashboard from your system tray to select a wallpaper!
- Drop any cool `.mp4` or `.html` file into the `papers/` folder and it will be available instantly on your dashboard!
